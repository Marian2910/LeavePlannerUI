# Introduction 
This application facilitates leave management and event planning within a company. Employees can log in to submit leave requests to HR, organize personal events, and view upcoming company-wide or personal events. HR personnel have additional functionalities, such as approving leave requests, scheduling events visible to all employees, and managing their personal events. Additionally, HR is responsible for managing the company's clients, including viewing contracts and adding or removing clients from the system.
# Getting Started
1.	Clone de repository: git clone ...
2.	Navigate to project directory: cd hr-fe-management-app
3.	Run npm install
4.	Run npm start or ng serve to run the app

This will start the development server. You can access the application at http://localhost:4200/ by default.
